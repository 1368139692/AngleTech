# 安装&使用(Install&Use)
### 前置插件(Reliance)
1. SlimeCustomizer 粘液自定义插件
2. GuizhanLibPlugin 鬼斩前置库
### 本体下载(Download)
1. GitHub
2. 群聊文件分享：895650188
### 安装方式(Install)
1. 安装前置插件，并启动服务端。
2. 下载插件数据，并将其替换至'plugins/SlimeCustomier'中，若安装有其他SlimeCustomier数据包，则将本数据包所有内容于原数据包合并。
3. 重启服务端
